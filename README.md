# Mod Revert Tool

這是一個使用 Electron 開發的 Windows 桌面應用程式，專門用來處理並還原遊戲中的模組設定檔。

## 🎯 專案目的

本專案的主要目的是為了**修復並還原《鳴潮 (Wuthering Waves)》因為 Mod Fixed 更新而遭到變動的 `mod.ini` 檔案**。當遊戲或外掛工具更新導致模組設定檔被覆蓋時，本工具可以幫助您快速找回並套用原本的設定檔。

## ✨ 核心功能

*   **自動掃描備份**：指定您的模組資料夾，程式會自動尋找所有包含 `bak`、`backup` 等備份附檔名的 `mod.ini` 檔案。
*   **視覺化批次管理**：透過圖形化介面，您可以清楚看到哪些檔案已經是最新狀態，並支援一鍵批次還原（自訂最新/最舊備份）。
*   **防呆檢測機制**：系統在背景會比對您的檔案內容，直接為您過濾掉已經還原過的設定檔，防止重複操作。
*   **流暢的使用體驗**：支援直接貼上路徑、即時操作與 Log 日誌追蹤，幫助您幾秒鐘內搞定所有 Mod 還原作業。

## 📝 備註

* 需要先安裝 [Node.js](https://nodejs.org/)。
* 執行 `npm start` 即可開啟工具介面。
