# Mod Revert Tool

這是一個使用 Electron 開發的 Windows 桌面應用程式，專門用來掃描與管理遊戲或模組資料夾中的 `mod.ini` 設定檔。如果你的模組設定檔被覆蓋或修改，這個工具可以幫助你快速掃描出所有含有備份檔（如 `.bak`, `.old`, `.orig` 等）的資料夾，並將其一鍵還原回 `mod.ini`。

## ✨ 主要功能

*   **自動掃描**：指定一個資料夾後，程式會自動遞迴掃描其下所有子資料夾。
*   **智慧識別**：自動尋找同時包含 `mod.ini` 與其備份檔案的目錄。支援識別 `bak`, `backup`, `.orig`, `.old` 等常見備份副檔名或命名方式。
*   **彈性還原**：如果有提供多個備份檔，可以透過下拉式選單自由選擇要還原使用哪一個版本的備份。
*   **一鍵操作**：支援「全選」與批次還原操作，省去一個一個資料夾進去替換的麻煩。
*   **即時日誌**：內建 Log 視窗，清楚顯示目前掃描與還原的進度、成功與失敗狀態。
*   **專屬客製化介面**：現代化的深色玻璃擬物風格（Glassmorphism）UI，具備精美動漫背景、自訂標題列與流暢的微動畫體驗。

## 🚀 開始使用

### 系統需求
*   [Node.js](https://nodejs.org/) (建議最新 LTS 版本)

### 安裝與執行

1.  **下載 / 複製此專案**至本機端。
2.  開啟終端機（Terminal）並切換至專案根目錄。
3.  **安裝依賴套件**：
    ```bash
    npm install
    ```
4.  **啟動應用程式**：
    ```bash
    npm start
    ```

## 🛠️ 使用說明

1.  開啟工具後，點擊畫面上方的 **「瀏覽」** 按鈕。
2.  選擇你存放模組（Mod）的主要根目錄。
3.  點擊 **「掃描」** 按鈕。程式便會在下方列表顯示所有找到可被還原的 `mod.ini` 項目。
4.  在列表中，如果你同一個目錄下有多個備份檔，可以在「備份」下拉選單中切換。
5.  勾選你想要進行還原的項目（或是直接點擊「全選」）。
6.  點擊紅色的 **「↩ Revert」** 按鈕開始還原。
7.  查看下方的 **Log 區塊**，確認所有檔案都已成功替換。

## 📂 專案架構

*   `main.js`：Electron 主進程（Main Process），負責建立視窗、資料夾讀取、檔案掃描與寫入操作（透過 IPC 通訊）。
*   `renderer.js`：前端邏輯（Renderer Process），負責處理 UI 互動、狀態更新與 Log 顯示。
*   `preload.js`：安全橋樑，將主進程的 API 暴露給前端呼叫。
*   `index.html`：前端介面架構，包含自訂標題列與玻璃風格排版。
*   `style.css`：前端介面樣式，包含深色主題、動畫與特效。
*   `package.json`：專案設定檔與依賴套件。
